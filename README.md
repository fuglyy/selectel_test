# SelectelTest

Чтобы запустить локальный сервер разработки, выполните команду:

```bash
ng serve
```

После запуска сервера откройте браузер и перейдите по адресу http://localhost:4200/.
Приложение будет автоматически перезагружаться при изменении исходных файлов.
